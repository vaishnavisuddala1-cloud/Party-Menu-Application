# Party Menu Application

A React + Vite party menu application with authentication, menu filtering, recipe saving, and responsive views.

## Installation

```bash
npm install
npm run dev
```

## Features

- Sign in flow with the provided authentication API
- Protected home route
- Menu filtering by category, diet, and search text
- Food detail page with ingredients and save toggle
- Saved recipes persisted in localStorage
- Responsive design using plain CSS

## Notes

The app reads the menu data from the provided JSON file in the workspace, without changing its structure.
